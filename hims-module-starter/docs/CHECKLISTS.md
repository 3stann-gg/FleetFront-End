# Checklists

## Pre-development checklist

- [ ] Starter folder copied / renamed  
- [ ] [MODULE-CONFIG.md](./MODULE-CONFIG.md) filled  
- [ ] HIMS UI Design Standard reviewed  
- [ ] Component library opened for reference  
- [ ] Shared CSS path verified  
- [ ] Theme boot script linked  
- [ ] Module name appears in sidebar brand  
- [ ] Sidebar menu items defined  
- [ ] `data-page` values planned  
- [ ] No Fleet-specific business copy left in templates  

## Pre-submission checklist

- [ ] All starter pages open without console errors  
- [ ] Light theme works  
- [ ] Dark theme works  
- [ ] Responsive layout checked (desktop / tablet / mobile)  
- [ ] Forms use shared form classes  
- [ ] Tables use `.fleet-table` (or approved table pattern)  
- [ ] Buttons use shared button classes  
- [ ] No duplicate component CSS systems  
- [ ] No inline business API secrets  
- [ ] Sample data clearly labeled as sample  
- [ ] README / config updated for the real module name  
- [ ] Keyboard basics checked (focus, labels)  
- [ ] Ready for Laravel data integration (UI stable)
