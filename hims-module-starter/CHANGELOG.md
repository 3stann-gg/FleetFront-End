# Changelog

All notable changes to the **HIMS Module Starter** package are listed here.

## [1.0.0] — 2026-07-20

### Added

- Initial module starter package structure
- Reusable layouts (main, dashboard, management, form, reports, settings, authentication)
- Starter pages (dashboard, management, create, edit, view, reports, analytics, settings, profile)
- Generic component placeholders
- Starter JavaScript architecture (no business logic / no API)
- Module CSS placeholder
- Documentation: setup, configuration, checklists
- Example: Patient Management Module demonstration

### Notes

- Compatible with Fleet reference design system in the parent repository
- Does not modify Fleet production frontend files
