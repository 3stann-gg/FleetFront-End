28. Responsive preview

Desktop (≥992px)

Fixed sidebar, full toolbars, multi-column stats.

Laptop (≤1199px)

Stats often 2 columns; navbar may wrap.

Tablet (≤991px)

Off-canvas sidebar; full-width main; table horizontal scroll.

Mobile (≤767px)

Single-column stats; stacked toolbars; tighter padding.

30. Component usage rules

Buttons

Use for actions. Primary = one main CTA per region.

Avoid multiple primaries competing.

Customize labels/icons only.

Tables

Use for record lists and dense data.

Avoid cards-only layouts for long lists without need.

Keep .fleet-table structure.

Modals

Use for focused create/edit/confirm flows.

Avoid multi-step wizards without design review.

Keep overlay + header/body/footer pattern.

Stats

Use for 3–6 KPIs at page top.

Avoid overcrowding the grid.

Customize icon, value, label only.
